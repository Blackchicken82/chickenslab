// Houdini VEX

VEX is Houdini's built-in shading and geometry language. It runs inside Wrangle nodes and lets you manipulate attributes, geometry and simulations at a low level.

// Example — Move points by noise

// Wrangle SOP — point mode
// Moves each point along its normal by a noise value

vector npos  = @P * 2.5;
float  amount = fit01( noise(npos), -0.3, 0.3 );

@P += @N * amount;
