// Houdini Python

Python in Houdini gives access to the full hou module — create nodes, set parameters, build HDAs and automate repetitive tasks directly from the shelf or a Python SOP.

// Example — Create a node and set parameters

import hou

# Get the /obj network
obj = hou.node("/obj")

# Create a new geometry node
geo = obj.createNode("geo", "my_geo")

# Create a sphere inside it
sphere = geo.createNode("sphere")
sphere.parm("radx").set(2.0)

# Layout the nodes neatly
geo.layoutChildren()
// result in network editor
